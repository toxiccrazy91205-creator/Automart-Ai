# AutoMart-AI
An end-to-end AI-powered retail analytics system that combines Machine
Learning, Intelligent Agents, and a Dashboard to generate actionable
business insights.

## 🚀 Features

-   Customer Segmentation (K-Means)
-   Inventory Management
-   Profit Analysis
-   Product Recommendations (Apriori)
-   Sales Forecasting (LSTM)
-   Streamlit Dashboard

## 🛠️ Tech Stack

-   Python, Pandas, NumPy
-   Scikit-learn, Mlxtend, PyTorch
-   Streamlit
-   LangChain + Gemini (optional)

## ▶️ Run Project

### Activate environment

    venv\Scripts\activate

### Install dependencies

    pip install -r requirements.txt

### Run dashboard

    streamlit run app/dashboard.py

### Run backend

    $env:PYTHONIOENCODING="utf-8"
    python main.py
