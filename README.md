# 🛒 Agentic AI-Powered Supermarket Analytics System

## 📌 Overview
This project is an AI-based decision support system designed for supermarkets.  
It uses machine learning and agent-based architecture to analyze sales, customers, and products, and generate actionable business insights.

---

## 🚀 Key Features
- 📈 Sales Prediction using LSTM (PyTorch)
- 👥 Customer Segmentation using K-Means
- 🛒 Product Recommendation using Apriori Algorithm
- 📦 Inventory Optimization & Restock Suggestions
- 💰 Profit & Loss Analysis
- 🎉 Festival-based Demand Analysis
- 🤖 Agent-based Decision System
- 📊 Interactive Dashboard using Streamlit

---

## 🧠 Project Architecture
1. **Data Collection & Preprocessing**: Collect and clean sales, customer, and product data.
2. **Model Training**: Train LSTM for sales prediction, K-Means for customer segmentation, and Apriori for product recommendations.
3. **Agent-based Decision System**: Implement agents to analyze insights and generate business recommendations.
